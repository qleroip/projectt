# RiskGuard release files

This folder contains ready-to-install demo builds.

## APK

- `apk/RiskGuard-Worker.apk` - Android application for the worker role.

## Desktop

- `exe/RiskGuard-Desktop-Setup.exe` - Windows installer for the expert desktop app.
- `exe/RiskGuard-Desktop-Installer.msi` - MSI installer for the expert desktop app.

For the demo, run the web server first, then install/open the desktop app or APK.
