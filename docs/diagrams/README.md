# RiskGuard diagrams

PlantUML diagrams for project documentation and presentation.

## Where to insert

| File | Diagram type | Recommended place |
| --- | --- | --- |
| `use-case.puml` | Use Case | After "Роли и сценарии использования" |
| `idef0-context.puml` | IDEF0 context A-0 | After "Архитектура системы" |
| `idef0-decomposition.puml` | IDEF0 decomposition A0 | After `idef0-context` |
| `sequence-risk-processing.puml` | Sequence | After "Описание системы" |
| `er-model.puml` | ER model | After "Модель данных" |
| `risk-lifecycle.puml` | State diagram | After "Жизненный цикл риска" |
| `cqrs-event-flow.puml` | CQRS/event flow | After "Фоновые задачи, события и CQRS" |
| `architecture.puml` | Component architecture | Documentation architecture section or presentation |

## How to export

Open any `.puml` file in PlantUML Online Server or a PlantUML plugin and export it as `PNG` or `SVG`.

Recommended for Word: `PNG`.
Recommended for presentation: `SVG` or high-resolution `PNG`.
